Baş səhifə

Hər kəs üçün əlçatan, pulsuz və hər yerdə mövcud olan intellektual miras insan növü kimi sağ qalmağın, inkişafın və genişlənmənin əsas şərti və təminatıdır.

Irrevion Science — elmi hesablamalar üçün tətbiqi alətlərin hazırlanmasına həsr olunmuş bir resursdur. Hal-hazırda veb sayt inkişafın ilkin mərhələsindədir.

Hazırda aşağıdakı layihələr mövcuddur və ya hazırlanma mərhələsindədir:
⚛️ irrevion/science — PHP-də yazılmış riyazi kitabxana;
🛠 IrryCMS — veb-saytın məzmununu idarə etmək üçün yüngül idarə paneli (PHP);
🧮 Science UI — API imkanlarını, məsələn, ⏲ fiziki ölçü vahidlərinin çeviricisini nümayiş etdirmək üçün front-end (React);
📡 Science API — fizika və riyazi hesablamaları yerinə yetirmək üçün API (PHP, Yii2);
🐧 Penguin@Iceberg — saf JavaScript ilə hazırlanmış balaca oyun, burada pinqvinin buz parçasından düşməməsini təmin etməlisiniz.
